Major developements:
1) If on task stages, 'Default In Projects' checkbox is ticked , then this task stage is added by defualt when any new project created and 'Use Default Swim Lanes' is checked to true.
2) Created new module to define type of project from options 'template', 'internal' and 'client'. When 'template' type of any project is created it can be further use to create new task stages and tasks when creating new project.
3) Added new filter in search view to search 'template' type of project in project module.

While creating new project:
1) If 'Use Default Swim Lanes' is checked to 'Yes' then when new project is created; all those 'task stages' which have 'Default In Projects' is true are added in project.
2) If template is selected, then the all 'tasks' from that template as well as 'task stages' which are there also added in that project.
