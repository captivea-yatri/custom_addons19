Payroll Accounting Extended

Overview

This app provide feature like
* Automated Reconciliation
* Direct Accounting Entry Visible
* Double Entry Book kipping for Payroll

How It Works

01

After installation app, The option to set partner with employee will get visible. Make sure you have set the partner properly.


02

Make sure proper contract and salary structure is configured. Create payslip with all those configuration.

after click the button compute sheet in header there are button create draft entry.

03

click the button create draft entry you will have an option to view Journal Entry and Journal Item related to payslip.

04

After clicking "Create Draft Entry" button, will create draft journal entry. Also have the PAY button visible.


05

As you can see this journal entry will have the partner and proper accounting entry. Post this JE.


06

After posting 1st JE, Come back to payslip and click on "PAY" button to make a payment. Click Create Payment.


07

Will make a payment and will also reconcile entry automatically, Also it will have proper label and partner on journal item.

As you can see this complete process behaves like double entry book kipping system.
