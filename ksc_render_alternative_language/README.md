================================================
Odoo 19 Render Alternative Language Translations
================================================

This module helps to render alternative language translation


============
Odoo Version
============
Odoo 19.0


============
Dependencies
============
1) base


=====================
Detailed Information
=====================
Follow steps to avail functionality:
(1) Go to Settings>Translations>Languages and open the form view of any language that you want to set alternative language to
(2) on the form view, you can see a field name "Alternative Language To Load Translation"
(3) select the language that you want as alternative to language of form view
(4) now, whenever for any block of website, if translation of main language is not available, it will first check if translation is available for alternative language and if it does, it will load that language for that block
(5) Also if the translation is not for alternative language it will load from default language.
