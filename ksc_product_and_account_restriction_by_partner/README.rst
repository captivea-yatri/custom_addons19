# Product & Account Restriction By Captivea

## Version
Odoo 19.0

## Dependencies
1. account
2. product

## Description
In this module, we created the kind of configuration specially for bill.
If bill is created or updated for the customer for which we configure the product and account restriction,
system allowed only those products and accounts which are configured for that customer.

## Features
Here are the main features of this module:
- **Product & Account Restriction:**
  A dedicated configuration object enables restriction management.
  Ensures that only predefined products and accounts can be used in bills for specific customers.
