

This module allows users to add attachments in sale quotation mail along with other details


**Table of contents**

.. contents::
   :local:

Configuration
=====
* Make sure sales module is installed.
* In main settings of sales make sure that pricelist option in 'pricing' section is marked true.

Usage
=====
Guidelines for use:

 * Create New Pricelist item or use exsiting one to add related pdf from extra prices button in product form view.
 * Create sale order and use particular pricelist where pdf document and products are already configured.
 * When click on 'SEND BY MAIL', the body of email will have attachment fetched from pricelist along with original file'.
