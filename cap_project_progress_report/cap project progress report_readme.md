=====================
Detailed Information
=====================
Follow steps to avail functionality cap project progress report:
Note===> .
(1) project => field=> 'Signatory Project Progress Report' you can see child customer and portal user of project customer.
(2) project => field=> 'Next Expected Project Progress Report Date' next project progress date which is configure field 'Report Frequency Days'.
(3) project => there is smart button project progress, you can create new project progress report.
(4) project progress report => create project progress report,
(5) project progress report=> manage domain history and past done task history,
(6) project progress report=>  button => send project progress report email to 'Signatory Project Progress Report' and request for sign 
(7) also you can edit / customize email template that is being sent to customer by accessing it through email templates
(8) button "Send the Project progress Report" will send report to customer with customized email template and also status will be change dynamically if customer signs report via email
