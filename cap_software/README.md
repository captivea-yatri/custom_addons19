====================================
Odoo 19 Cap Software
====================================

This Module extends the functionality of odoo's Projects module and CRM Module, in which cap software makes available the option for selecting which software and its version the project or opportunity is related to


============
Odoo Version
============
Odoo 19.0


============
Dependencies
============
Following Modules must be already installed to have Cap Marketing CRM Automation functionality
1) Projects
2) CRM
3) base



=====================
Detailed Information
=====================
Follow steps to avail functionality:
(1)Go to apps and install Cap Software
(2)Go to projects app, in Top menu bar select configuration>Software and specify all the soft wares as per your company requirements
(3)Go to projects app, in Top menu bar select configuration>Software Version and specify all the soft wares and its versions as per your company requirements

after completing above configuration, your specified software versions are available for selection in project form view and CRM opportunity form view as well
