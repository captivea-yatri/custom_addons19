================================================

Custom Module: cap_critical_schedule_action

============
Odoo Version
============
Odoo 19.0

============
Dependencies
============
Following Modules must be already installed.

=> 'base','base_setup','mail'

=====================
Functionalities
=====================
(1) maintaining cron failure log with detailed information and time at which it gets stuck
(2) sends mail to concerned users (can configure them in general settings) about cron failure with detailed information
